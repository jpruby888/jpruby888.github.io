Linux安装部署上线项目流程

发表于2020-12-25|更新于2022-03-17
|阅读量:

环境安装

一般在linux上安装软件有三种方式：

  • rpm
  • 解压缩
  • yum在线安装

JDK安装（使用rpm方式）

  • 下载jdk的rpm包
  • 通过xftp上传到服务器
  • 安装java环境

检查是否存在java环境: java -version
检查jdk版本信息: rpm -qa|grep jdk
卸载jdk: rpm -e –nodeps jdk……
安装jdk: rpm -ivh jdk-….rpm

  • 发布项目
    将springboot项目打包生成jar包，并放到linux服务器上

  • 开启项目对应端口号(以9000为例)

    firewall-cmd –zone=public –add-port=9000/tcp –permanent

  • 重启防火墙

    systemctl restart firewalld.service

  • 查看防火墙端口号是否打开

    firewall-cmd –list-ports

  • 阿里云需要配置安全组规则


  • 赋予文件权限(读、写、可执行)

    chmod 777 springboot-web.jar

  • 执行文件

    java -jar springboot-web.jar


执行成功，即可从公网访问网站。

Tomcat安装（使用解压缩方式）

ssm项目中war包需要放到Tomcat中运行！

  • 下载Tomcat
    apache-tomcat-XXX.tar.gz
    Tomcat 9 链接

  • 解压文件

    tar zxvf apache-tomcat-9.0.37.tar.gz

  • 进入Tomcat的bin目录下，启动Tomcat测试

    启动：./startup.sh
    结束：./shutdown.sh

Docker安装（使用yum方式安装）

需要在联网环境下进行，需要从网上下载文件。

  • 检测CentOS 7

    cat /etc/redhat-release
    CentOS Linux release 7.7.1908 (Core)

  • 安装准备环境

    yum -y install 包名
    yum -y install gcc
    yum -y install gcc-c++

  • 卸载以前版本，根据官网依次执行指令安装。

使用宝塔面板安装软件

打开宝塔面板可视化工具，进行可视化操作即可。
